# Subscription vs. API
  - subscription
    - fixed price
    - multiple price tiers, $20, $100, $200 
    - daily/hourly limits based on subscription price
    - typically used as a tool for development, imaging, and other direct activities
  - API
    - pay as you go
    - per token in and per token out
    - usually more expensive thatn subscriptions
    - limited by how much you want to spend
    - Auth Token and Endpoing
    - can be used a tool, but more fit for supporting AI in an app
